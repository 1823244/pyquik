PyQuik

Tranding with python and Quik

Fork me on Github

Features

  • Import data with DDE
  • Submit orders with trans2quik.dll
  • Native interface with TA-Lib (more then 150 functions and indicators)
  • Monitoring over WEB interface

Example

Sample trading robot


from quik import Quik
from trading import order

# Called every price change
def handle_tick( ticker ):
    if ticker.price < 100.0:
        ticker.buy( order.MARKET_PRICE, 1 )
    if ticker.price > 200.0 and ticker.orders.quantity > 0:
        ticker.sell( order.MARKET_PRICE, ticker.orders.quantity )

quik = Quik("c:\\quik","QuikDDE")
quik.SBER03.ontick( handle_tick )
quik.run()
